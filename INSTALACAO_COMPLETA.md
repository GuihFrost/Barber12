## 📦 Como usar o arquivo ZIP:

### 1. Extrair o arquivo
```bash
unzip BarberTime-Complete.zip
cd BarberTime-Complete
```

### 2. Instalação automática
```bash
chmod +x install.sh
./install.sh
```

### 3. Executar o sistema
```bash
# Terminal 1 - Backend
php artisan serve

# Terminal 2 - Frontend (opcional para desenvolvimento)
npm run dev
```

### 4. Acessar
- **Sistema**: http://localhost:8000
- **API**: http://localhost:8000/api/health
